# Paginate
Paginate provides a generic way to show a section of an array, useful for creating components such as pagination or sliders.
 
## Base
The most concise way of accessing the pagination state is to use Render Props (i.e a function as a child).

<Preview>
  <Story name='Base'>
    <Paginate>
      {({ items, current, setCurrent }) => (
        <Fragment>
          <div>Current: {current}</div>
          {Array.from(Array(6).keys()).map((x, i) => (
            items.includes(i) && <div key={`item-${i}`}>Item {i}</div>
          ))}
          <Button type='Primary' disabled={current === 0} onClick={() => setCurrent(current - 1 < 0 ? 0 : current - 1)}>Prev</Button>
          <Button type='Primary' disabled={current === 5} onClick={() => setCurrent(current + 1 > 5 ? 5 : current + 1)}>Next</Button>
        </Fragment>
      )}
    </Paginate>
  </Story>
</Preview>

## Context
Alternatively, you can use `useContext` to access the pagination state.

<Preview>
  <Story name='Context'>
    {() => {
      const App = () => {
        const { items, current, setCurrent } = useContext(PaginateContext)
        return (
          <Fragment>
            <div>Current: {current}</div>
            {Array.from(Array(6).keys()).map((x, i) => (
              items.includes(i) && <div key={`item-${i}`}>Item {i}</div>
            ))}
            <Button type='Primary' disabled={current === 0} onClick={() => setCurrent(current - 1 < 0 ? 0 : current - 1)}>Prev</Button>
            <Button type='Primary' disabled={current === 5} onClick={() => setCurrent(current + 1 > 5 ? 5 : current + 1)}>Next</Button>
          </Fragment>
        )
      }
      return (
        <Paginate>
          <App />
        </Paginate>
      )
    }}
  </Story>
</Preview>

## Limit
Set the `limit` prop and adjust the current index to show more items per page. This example has a limit of 3 per page.

<Preview>
  <Story name='Limit'>
    <Paginate limit={3}>
      {({ items, current, setCurrent }) => (
        <Fragment>
          <div>Current: {current}</div>
          {Array.from(Array(6).keys()).map((x, i) => (
            items.includes(i) && <div key={`item-${i}`}>Item {i}</div>
          ))}
          <Button type='Primary' disabled={current === 0} onClick={() => setCurrent(current - 3 < 0 ? 0 : current - 1)}>Prev</Button>
          <Button type='Primary' disabled={current === 6} onClick={() => setCurrent(current + 3 > 5 ? 3 : current + 1)}>Next</Button>
        </Fragment>
      )}
    </Paginate>
  </Story>
</Preview>

## Infinite
Simply set the current page back to the start for an infinite loop.

<Preview>
  <Story name='Infinite'>
    <Paginate limit={1}>
      {({ items, current, setCurrent }) => (
        <Fragment>
          <div>Current: {current}</div>
          {Array.from(Array(6).keys()).map((x, i) => (
            items.includes(i) && <div key={`item-${i}`}>Item {i}</div>
          ))}
          <Button type='Primary' onClick={() => setCurrent(current - 1 < 0 ? 5 : current - 1)}>Prev</Button>
          <Button type='Primary' onClick={() => setCurrent(current + 1 > 5 ? 0 : current + 1)}>Next</Button>
        </Fragment>
      )}
    </Paginate>
  </Story>
</Preview>
