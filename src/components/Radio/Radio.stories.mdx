# Radio
A simple Radio to allow users to carry out an action. This is a controlled component so a `value` and an `onChange` prop must be passed.

## Base
For use to indicate a boolean value.

<Preview>
  <Story name="Base">
    {() => {
      const [value, setValue] = useState(false)
      return (
        <div>
          <Text>Value: {`${value}`}</Text>
          <Radio id="radio" value={value} onChange={setValue} />
        </div> 
      )
    }}
  </Story>
</Preview>

## Group
For use to indicate multiple boolean values.

<Preview>
  <Story name="Group">
    {() => {
      const [value, setValue] = useState(null)
      return (
        <div>
          <Text>Value selected: {JSON.stringify(value)}</Text>
          <Radio id="radio" value={value === 'cat'} onChange={(value) => setValue('cat', value)} />
          <Radio id="radio" value={value === 'dog'} onChange={(value) => setValue('dog', value)} />
          <Radio id="radio" value={value === 'horse'} onChange={(value) => setValue('horse', value)} />
        </div>
      )
    }}
  </Story>
</Preview>
