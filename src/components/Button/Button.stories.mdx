# Button
A simple button for allowing users to carry out any necessary action(s).

This is the button in its simplest form, using all default props.

## Variants
Pass a value to the `variant` prop and you'll see the desired button rendered.

`Primary` is the default value, so the prop can be omitted.

### Primary
<Preview>
  <Story name='Primary'>
