# Button
A simple buttons to allow users to carry out an action.
 
## Primary
For use as a main call to action.

<Preview>
  <Story name='Primary'>
    <Button>Primary Button</Button>
  </Story>
</Preview>

## Secondary
Secondary buttons should be used sparingly, in instances where a primary button would have too much emphasis.

<Preview>
  <Story name='Secondary'>
    <Button variant='Secondary'>Secondary Button</Button>
  </Story>
</Preview>

## Tertiary
Tertiary buttons should be used in conjunction with primary buttons where there are two actions.

<Preview>
  <Story name='Tertiary'>
    <Button variant='Tertiary'>Tertiary Button</Button>
  </Story>
</Preview>

## Link
If the `href` prop is supplied, the button will render correctly as an `<a>` tag.

  <Story name='Link'>
    <MemoryRouter>
      <Button href='/'>Link Button</Button>
    </MemoryRouter>
  </Story>
</Preview>
