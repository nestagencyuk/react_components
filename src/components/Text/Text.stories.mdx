# Text
A text component which renders a text tag. Styles are decoupled from markup so using a `h1` tag alongside `Intro` variant could be a valid use case.
 
## Variants
Pass a value to the `tag` prop to alter the HTML Tag rendered.

`p` is the default value, so the prop can be omitted.

Pass a value to the `variant` prop to alter the CSS used to style the tag.

`P` is the default value, so the prop can be omitted.

## Alignment
Pass a value to the `align` prop and you'll see the desired text rendered.

`Left` is the default value, so the prop can be omitted.
