# Header
A simple header component, used in composition with other components e.g. [Modal](?path=/docs/components-base-modal--base).

> **NOTE:** For a top-level page header, see [Page Header](/?path=/docs/layout-page-header--base)

## Sizes
Pass a value to the `size` prop and you'll see the desired header rendered.

`Medium` is the default value, so the prop can be omitted.
