# Drawer

A simple drawer that can be used to show/hide content on a specified side of the browser.

## Base
This is the drawer in its simplest form.

<Preview>
  <Story name='Toggle'>
    {() => {
      const [toggled, setToggled] = useToggle()
      return (
        <Fragment>
          <Button onClick={() => setToggled(!toggled)}>Open Drawer</Button>
          {toggled && (
            <Drawer onClick={() => setToggled(false)}>
              <Text>
                Item 1
              </Text>
                <Text>
                Item 2
              </Text>
            </Drawer>
          )}
        </Fragment>
      )
    }}
  </Story>
