# Drawer

A simple drawer that can be used to show/hide content on a specified side of the browser.

<Preview>

              <Drawer open={toggled} onClick={() => setToggled(false)}>
                <Text>
                  Item 1
                </Text>
                  <Text>
                  Item 2
                </Text>
              </Drawer>
          }
        </Fragment>
      )
