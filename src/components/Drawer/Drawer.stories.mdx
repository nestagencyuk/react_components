# Drawer
A simple drawer that can be used to show/hide content on a specified side of the browser. Compatible with the default state machine.

## Base
This is the drawer in its simplest form.

<Preview>
  <Story name="Toggle">
    {() => {
      const [state, dispatchEvent] = useStateMachine()
      return (
        <Fragment>
          <Button onClick={() => dispatchEvent('OPEN')}>Open Drawer</Button>
          <Drawer state={state} onClick={() => dispatchEvent('CLOSE')}>
            <Text>
              Item 1
            </Text>
              <Text>
              Item 2
            </Text>
          </Drawer>
        </Fragment>
      )
    }}
