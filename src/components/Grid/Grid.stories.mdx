# Grid
A simple CSS grid system. The basic system comprises of a 12 column grid. The grid is however not reliant on multiple rows so is a little lighter when it comes to outputted markup.

## Gutter

By default the grid has no gutter, but this can be added via the 'gutter' prop:

<Preview>
  <Story name='Gutter'>

## Alignment
The align prop aligns content within a `GridItem`. Simple pass an object with an `x` or `y` value, or both.

### Alignment along the X Axis

  <Story name='XAlignment'>
