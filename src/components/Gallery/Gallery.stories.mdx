## Lightbox
To create a lightbox effect, pass the `lightbox` boolean prop. Your items array must then be a function that returns an array. You can then access `handleClick` and pass it to a button within each gallery item to trigger opening the lightbox.

<Preview>
  <Story name='Lightbox'>
    <Gallery lightbox items={({ handleClick }) => [
      <div className="interactive" onClick={() => handleClick(0)}>
