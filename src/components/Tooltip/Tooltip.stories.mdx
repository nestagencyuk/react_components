## Alignment
Pass a value to the `align` prop and you'll see the desired tooltip rendered. `Top` is the default value, so the prop can be omitted.

### Left
<Preview>
  <Story name="Align-Left">
    <Tooltip align="Left" render={<div>I am a Tooltip!</div>}>
      {(value) => (<RefButton {...value} >Align left</RefButton>)}
    </Tooltip>

</Preview>
