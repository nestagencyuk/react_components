# Tooltip
A simple tooltip for rendering information to your users.

## Base
This is the tooltip in its simplest form.

<Preview>
  <Story name="Base">
    {() => {
      const [el, setEl] = useState()
      const ref = useRef()
      useEffect(() => {
        setEl(ref.current)
      }, [])
      return (
        <Text>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit, <strong ref={ref} style={{ display: 'inline-block'}}>toggle</strong> do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
          <Tooltip attachTo={el} align="Top">
            I am a Tooltip!
          </Tooltip>
        </Text>
      )
    }}
  </Story>
</Preview>

## Variants
Pass a value to the `align` prop and you'll see the desired tooltip rendered.

`Left` is the default value, so the prop can be omitted.

<Preview>
  <Story name="Align-Left">
    {() => {
      const [el, setEl] = useState()
      const ref = useRef()
      useEffect(() => {
        setEl(ref.current)
      }, [])
      return (
        <Text>
          <strong ref={ref} style={{ display: 'inline-block'}}>Align left</strong>
          <Tooltip attachTo={el} align="Left">
            I am a Tooltip!
          </Tooltip>
        </Text>
      )
    }}
  </Story>
</Preview>
