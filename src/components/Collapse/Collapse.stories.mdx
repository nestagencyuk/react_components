# Collapse
A simple collapsible panel can be built using the [Toggle](/?path=/docs/hooks-toggle--base) and [ToggleGroup](/?path=/docs/hooks-toggle-group--base) interaction helpers.
 
## Base
A single toggleable panel using [Toggle](/?path=/docs/hooks-toggle--base).

<Preview>
  <Story name='Base'>
    {() => {
      const [toggled, setToggled] = useToggle()
      return (
        <Collapse open={toggled} header={{ heading: 'A heading' }} onToggle={() => setToggled(!toggled)}>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. 
        </Collapse>
      )
    }}
  </Story>
</Preview>

## Multiple
[ToggleGroup](/?path=/docs/hooks-toggle-group--base) can be used to create an accordion effect.

<Preview>
  <Story name='Multiple'>
    {() => {
      const [toggles, setToggled] = useToggleGroup()
      return (
        <Fragment>
          <Collapse 
            open={toggles['one']} 
            header={{ heading: 'A heading' }} 
            onToggle={() => setToggled('one')}
          >
            Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. 
          </Collapse>
          <Collapse 
            open={toggles['two']} 
            header={{ heading: 'Another heading' }} 
            onToggle={() => setToggled('two')}
          >
            Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
          </Collapse>
          <Collapse 
            open={toggles['three']} 
            header={{ heading: 'Another heading' }} 
            onToggle={() => setToggled('three')}
          >
            Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
          </Collapse>
        </Fragment>
      )
    }}
  </Story>
</Preview>

## Nested
[ToggleTree](/?path=/docs/hooks-toggle-tree--base) can be used to create a nested accordion effect.
