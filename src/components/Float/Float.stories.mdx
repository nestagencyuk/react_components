## Portal
The `portal` prop is perfect for creating content that renders over the main page body - e.g Alerts, Modals.

<Preview>
  <Story name='Portal'>
    <Toggle>
      {({ toggled, setToggled }) => (
        <Fragment>
          <Button type={'Primary'} onClick={() => setToggled(!toggled)}>{toggled ? 'Close' : 'Toggle'} Float</Button>
          {toggled && <Float portal>
            I am floating!
          </Float>}
        </Fragment>
      )}
    </Toggle>
  </Story>
</Preview>
