## Animated

Animate values in or out, useful for showing loading or timeout states.

<Preview>
  <Story name="Animate Up">
    {() => {
      const startingValue = 0;
      const [value, setValue] = useState(startingValue)
      const countDown = () => {
        setValue(100 + startingValue)
      }
      
      return (
        <Fragment>
          <Button className="m--b-sm" onClick={() => countDown()}>
            Animate in
          </Button>
          <ProgressBar width={`${value}%`} />
        </Fragment>
      )
    }}
  </Story>
</Preview>
