## Animated
Use state to animate the bar in or out.

<Preview>
  <Story name="Animate-Up">
    {() => {
      const startingValue = 0;
      const [value, setValue] = useState(startingValue)
      const animate = () => {
        setValue(100 + startingValue)
      }
      return (
        <Fragment>
          <ProgressBar className="m--b-sm" value={value} />
          <Button onClick={() => animate()}>
            Animate in
          </Button>
        </Fragment>
      )
    }}
  </Story>
</Preview>
