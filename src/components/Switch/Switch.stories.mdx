# Switch
A simple Switch which allows users to carry out an action. This is a controlled component so a `value` and an `onChange` prop must be passed.

## Base
The switch in its simplest form.

<Preview>
  <Story name="Base">
    {() => {
      const [value, setValue] = useState(false)
      return (
        <div>
          <Text>Value: {`${value}`}</Text>
          <Switch id="switch" value={value} onChange={setValue} />
        </div>
      )
    }}
  </Story>
</Preview>

## Disabled
Pass boolean `true` to the `disabled` prop to render the desired switch. `false` is the default value, so the prop can be omitted.

<Preview>
  <Story name="Disabled">
    <Switch id="switch" disabled />
  </Story>
</Preview>
