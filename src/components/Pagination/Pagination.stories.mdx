# Pagination

A simple pagination component.

## Base

This is the pagination in its simplest form

<Preview> 
  <Story name='Base'>
    <Pagination
      current={2}
      items={[0, 1, 2, 3, 4]}
      onPrev={() => alert('Prev')}
      onCurrent={(i) => alert(`Current ${i}`)}
      onNext={() => alert('Next')}
    />
  </Story>
</Preview>

## Variants

Pass a value to the `variant` prop and you'll see the desired pagination rendered.

`Numbers` is the default value, so the prop can be omitted.

<Preview>
  <Story name='Dots'>
    <Pagination variant='Dots' current={2} items={[0, 1, 2, 3, 4]} onCurrent={(i) => alert(`Current ${i}`)} />
  </Story>
</Preview>
