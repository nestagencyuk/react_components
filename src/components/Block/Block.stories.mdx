# Block
A block component that renders any optional combination of an image, heading, body text or button.

## Base
This is the block using all optional props.

<Preview>
  <Story name='Base'>
    <MemoryRouter>
      <Block image={{ aspect: '1x1', align: 'Start', src: 'https://source.unsplash.com/random/1000x1000', alt: 'Block Image'}} header={{ heading: 'Heading', subheading: 'Sub Heading'}}>
        <Text>
          Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
        </Text>
      </Block>
    </MemoryRouter>
  </Story>
</Preview>

## Variants
Pass a value to the `size` prop and you'll see the desired block rendered.

`Medium` is the default value, so the prop can be omitted.

## With Button
