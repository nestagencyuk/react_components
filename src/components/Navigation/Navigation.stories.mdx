# Navigation
A simple inline hyperlink navigation with the option to pass in your own link component (e.g Gatsby link or React Router link).
 
## Base
A common navigation style.

  <Story name='Base'>
    <MemoryRouter>
      <Navigation 
        brand={{ align: 'Center', image: { aspect: '1x1', src: logo, alt: 'My Company Logo'} }}
