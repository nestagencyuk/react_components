# Slider
A slider for displaying a series of content. Takes an `items` array of components which can contain whatever you want. 
 
## Base

The slider in its simplest form.

<Preview>
  <Story name='Base'>
    <Slider items={[
      <Box className='p--md bg--ui-one'><div>Slide 1</div></Box>,
      <Box className='p--md bg--ui-two'><div>Slide 2</div></Box>,
      <Box className='p--md bg--ui-one'><div>Slide 3</div></Box>,
      <Box className='p--md bg--ui-two'><div>Slide 4</div></Box>
    ]} />
  </Story>
</Preview>

## Variants
Pass a value to the `variant` prop and you'll see the desired slider rendered.

`Slide` is the default value, so the prop can be omitted.

<Preview>
  <Story name='Fade'>
    <Slider variant='Fade' items={[

</Preview>
