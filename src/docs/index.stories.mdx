# Introduction

This repository contains Nest's React components for use across the business. The main purpose of this project is to provide a single source of truth of components for use internally and across all client projects.

## Requirements: 
- [Node.js](https://nodejs.org/) v9+
- [React.js](https://reactjs.org/) v16+
- [Sass](https://www.npmjs.com/package/sass) v1.25+ (for component style modules only)

## Usage:
Install:
```sh
$ npm i nest-components
```

### JavaScript Only
Import the desired components into your .jsx or .tsx files. This will include the components in their unstyled format. To include styles, read our [style docs](/?path=/docs/getting-started-sass-modules--page)

```js
import { Button } from 'nest-components'

ReactDOM.render(<Button />, document.getElementById('app'))
```
