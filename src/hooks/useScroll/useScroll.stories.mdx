# Scroll
Listen to a scroll event but using requestAnimationFrame for best performance.

## Base
Basic scroll x and y values.

<Preview>
  <Story name="Base">
    {() => {
      const [scroll] = useScroll()
      return (
        <div style={{ height: '1000px' }}>
          <h4>Scroll X:</h4>
          <p>{`${scroll.x}`}</p>
          <h4>Scroll Y:</h4>
          <p>{`${scroll.y}`}</p>
        </div>
      )
    }}
  </Story>
</Preview>

## Acceleration
Pass a decimal to add an accelation/deceleration effect.
