# Toggle Group
A toggle group is similar to the singular [Toggle](/?path=/docs/hooks-toggle--base) however it allows for multiple toggleables in the same set. Useful for tabs, accordions and other similar hidden content.

## Base
The `useToggleGroup` hook can be used to switch toggled state.

<Preview>
  <Story name='Base'>
    {() => {
      const [toggles, setToggled] = useToggleGroup()
      return (
        <Fragment>
          <Button className='m m--r-md' onClick={() => setToggled('one')}>{toggles['one'] ? 'Untoggle' : 'Toggle me!'}</Button>
          <Button className='m m--r-md' onClick={() => setToggled('two')}>{toggles['two'] ? 'Untoggle' : 'Toggle me!'}</Button>
          <Button onClick={() => setToggled('three')}>{toggles['three'] ? 'Untoggle' : 'Toggle me!'}</Button>
        </Fragment>
      )
    }}
  </Story>
</Preview>

## Multi
Have multiple toggles at once.
