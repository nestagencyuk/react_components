# Toggle
A toggle hook that provides a generic boolean state that is accessible by any component that is a descendent in the component tree. Useful for modals, alerts and other hidden content where local state wouldn't be appropriate. For toggling multiple items, see [ToggleGroup](/?path=/docs/hooks-toggle-group--base).
 
## Base
The hook in its simplest form.

<Preview>
  <Story name="Base">
    {() => {
      const [toggled, setToggled] = useToggle()
      return (
        <Button onClick={() => setToggled(!toggled)}>{toggled ? 'Untoggle' : 'Toggle me!'}</Button>
      )
    }}
